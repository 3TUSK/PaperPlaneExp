# 纸飞机制作说明

  1. 准备一张 A4 大小的纸。  
     ![Step1](Step1.svg)
  2. 将纸上方一侧的两个角朝里对折，使得两个角在中间相遇。  
     ![Step2](Step2.svg)
  3. 将两个角再折一下，使两个角在中间相遇。  
     ![Step3](Step3.svg)
  4. 现在的两个角看起来更像是两条边。  
     将这两条边继续向中间对折。  
     ![Step4](Step4.svg)
  5. 将现在的整个折纸翻过去，然后从右向左对折。  
     ![Step5](Step5.svg)
  6. 捏住整个折纸的中间，然后将两侧的纸各展开 90 度。
     你的纸飞机现在已经可以飞了。  
     ![Step6](Step6.svg)
